Cara membuat akun blog

Membuat blog di Medium.com

View on GitHub Download .zip Download .tar.gz

1. Klik tombol Sign up pada pojok kanan atas

2. Pilih metode pembuatan akun

Medium.com menyediakan berberapa opsi untuk membuat akun blog. Saya akan mencotohkan bagaimana membuat akun blog menggunakan akun google.

3. Klik pilihan continue with Google

Masukan username dan password anda, selanjutnya klik allow Selanjutnya Klik tombol Create Account

4.Klik topik kesukaan anda

Medium akan menaampilkan feeds berdasarkan topik yang anda pilih

5. Untuk menulis post, anda bisa mengetik di bagian yang terdapat tulisan "write here"

jika ingin menambahkan gambar anda bisa menggunakan mode full screen di bagian "write here"

6. Untuk menulisakan komentar, anda bisa mengklik logo komentar setelah post sudah di publish
